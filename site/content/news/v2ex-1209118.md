GCP 、Azure 同理，它们也可以用 Anthropic 的模型，但是前些天 AWS 出了这么一篇新闻：
https://aws.amazon.com/claude-platform/
大概就是说，可以通过 AWS 访问 Anthropic 官方 API ，同时使用 AWS 的鉴权认证、账单付款。

（但有一说一 Anthropic 官方 API 可用性不咋地就是了……）
PS：
Anthropic 官方 API 有一些特性是没开放给第三方云厂商的，比如 Opus 4.6 Fast Mode 、比如 Advisor Tool……官方文档上都有写，如果某家宣称是官方 API ，可以问问看是否支持这些特性。
