OpenAI 和微软再次修改了双方的合作协议。表面上看，这是 OpenAI 摆脱微软独家约束的一步；但从新条款看，双方都拿到了自己最需要的东西：OpenAI 获得更多云分发自由，微软则锁定了长期授权和 Azure 大客户关系。

最重要的变化是，微软对 OpenAI 模型和产品 IP 的授权从过去近似“直到 AGI 出现”的模糊绑定，改为持续到 2032 年的非独家许可。微软仍被称为 OpenAI 的主要云合作伙伴，OpenAI 也承诺在 10 月购买额外 2500 亿美元 Azure 云服务，这对微软股东是一个明确的续约信号。

但 OpenAI 现在可以通过任意云服务商向客户提供所有产品。公告中仍写着产品会“优先”在 Azure 上提供，除非微软无法或选择不支持所需能力；不过“优先”的排他程度并未被完全说明。

这项调整解决的是 OpenAI 与 Amazon 合作带来的潜在法律风险。今年 2 月，Amazon 宣布最多向 OpenAI 投入 500 亿美元，双方还计划在 AWS Bedrock 上共同开发有状态运行时技术。如果旧协议仍赋予微软更强独占权，微软理论上可能围绕这笔 AWS 合作发起法律挑战。

新协议把边界重新画清：微软继续享有长期 OpenAI 技术授权和 Azure 主要合作地位，OpenAI 则获得跨云销售和建设基础设施的空间。对一家算力需求极高、又准备继续扩张的 AI 公司来说，这种弹性比单一云绑定更关键。
