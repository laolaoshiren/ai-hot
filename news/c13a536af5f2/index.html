百度文库网盘发布GenFlow 4.0：月活用户超1亿，要把网盘变成全端AI工作台

⚡ 一句话看懂：百度文库网盘发布GenFlow 4.0：月活用户超1亿，要把网盘变成全端AI工作台。来源：InfoQ AI。

4 月 27 日，百度文库网盘联合发布通用智能体 GenFlow 4.0，并全面升级 Office Agent。

新版本主打两个方向：一是让个人用户在电脑和手机上都能使用 AI 完成办公、学习和内容创作；二是让团队用户把 AI Agent 直接接入网盘里的数据，形成一个可协同工作的“AI 工作台”。

据介绍，GenFlow 4.0 已经拥有超过 1 亿月活用户，月任务交付量达到 2 亿次。

百度网盘还将在百度 Create 大会上发布视频剪辑 Agent 能力，并计划在 5 月底推出团队版“Agent 协作军团”。

这意味着，百度文库网盘不再只是文件存储和资料查询工具，而是试图把文档、表格、PPT、图片、视频、团队资料和 AI Agent 放到同一个工作流里。

Office Agent 升级：一句话调动 PPT、Excel、Word Agent 这次升级中，Office Agent 是重点。

百官方表示，Office Agent 已经可以并行调用 PPT、Excel、Word Agent，支持更多模态内容理解和生成，也支持更自由的元素级编辑。

它的目标不是简单“生成一份文档”，而是尽量把办公内容做到“直接可用”。

在 PPT 场景中，GenFlow 4.0 可以生成专业汇报、行业报告、品牌宣贯、卡通故事等多种类型 PPT。

它不只负责排版，还能完成素材整理、逻辑串联、结构化图表生成和视觉美化。

对于老旧 PPT 或草稿 PPT，也可以一键美化，在保留内容的基础上提升表达和视觉效果。

Excel Agent 则主打自然语言处理表格。

用户可以直接输入需求，比如“识别所有实验数据文件并汇总成新表，结果保留三位小数，用红色标注异常数据”，或“统计公司营业额每个季度的增长率”“根据以上数据做销售趋势折线图”。

系统可完成数据筛选、表格生成、公式处理、图表制作等操作，减少人工复制粘贴和复杂公式编辑。

Word Agent 则面向长文生成和编辑，支持快速生成市场调研、学术报告等专业长文。

在文库网盘学术数据能力加持下，它可以自动感知和抓取行业数据，生成结构化图表，并完成排版美化和全文编辑，让文档从“写完”更接近“可交付”。

除了办公能力，GenFlow 4.0 还升级了 AI 练题、AI 看点、AI 求职等学习能力，以及 AI 视频创作、AI 照相馆、AI 相簿等生活娱乐功能。
